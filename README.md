# brickmmo-gps

A GPS system for a LEGO™ city using LEGO™ bricks and a Pixy2 AI camera. 

## Requirements:

* [BrickMMO](https://www.brickmmo.com/)
* [BrickMMO SSO](https://sso.brickmmo.com/)
